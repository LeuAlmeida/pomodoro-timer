<h3>:electric_plug: Requeriments</h3>

* NodeJS >= 10.16
* Yarn >= 1.21

<h3>:arrows_counterclockwise: Running the application</h3>

```bash
# Clone the project
$ git clone https://github.com/LeuAlmeida/pomodoro-timer.git

# Enter the folder
$ cd pomodoro-timer

# Install the dependencies
$ yarn

## Start the project
$ yarn start
```
